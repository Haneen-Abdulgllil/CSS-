Box Shadow Properties

This is a div element with a box-shadow:
box-shadow: 10px 10px;
This is a div element with a box-shadow:
box-shadow: -10px -10px;
box-shadow with color:

box-shadow: 10px 10px grey;
box-shadow with blur effect:

box-shadow: 10px 10px 5px grey;
box-shadow with blur effect from four sides:

box-shadow: 0px 0px 10px grey;
box-shadow with spread radius:

box-shadow: 10px 10px 5px 12px grey;
box-shadow with inset parameter:

box-shadow: 10px 10px 5px grey inset ;
Multiple shadows:

box-shadow: 5px 5px blue, 10px 10px red, 15px 15px green;
Multiple shadows:

box-shadow: 5px 5px 8px blue, 10px 10px 8px red, 15px 15px 8px green;
box-shadow with none parameter:

box-shadow: none ;
box-shadow with initial parameter:

box-shadow: initial ;
box-shadow with inherit parameter:
box-shadow: inherit ;

Relative Units

em

Relative to the font-size of the element (2em means 2 times the size of the current font)

ex

Relative to the x-height of the current font (rarely used)

ch

Relative to the width of the "0" (zero)

rem

Relative to font-size of the root element

vw

Relative to 1% of the width of the viewport*

vh

Relative to 1% of the height of the viewport*

vmin

Relative to 1% of viewport's* smaller dimension

vmax

Relative to 1% of viewport's* larger dimension

%

Relative to the parent element

Absolute Units

cm

centimeters

mm

millimeters

in

inches (1in = 96px = 2.54cm)

px

pixels (1px = 1/96th of 1in)

pt

points (1pt = 1/72 of 1in)

pc

picas (1pc = 12 pt)
